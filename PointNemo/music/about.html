What is Sonar?

Sonar is a service developed with the sole goal of helping music lovers find their sound. We use listening data collected via their chosen streaming service's API, and configure them a list of songs and artists they may like.

How we built Sonar

Sonar was built using the standard web languages (HTML5, CSS3 and JavaScript), with the SCSS preprocessor for the CSS. We used Github to keep all of our code online, and as a version control technique to track our changes.

Why we built Sonar?

We built Sonar as we believed music streaming services weren't doing a good enough job at providing music listeners with music they actually want to listen to, and didn't attempt to open their eyes to new artists, and music in their favourite genres.

Who built Sonar?

Sonar was built by 2 students who are new to the world of web development. Jayden and Aiden found a passion for web development through college, and wanted to collaborate on a project that could serve a purpose for people, while also being related to music, which everyone loves.

Where is Sonar hosted?

Sonar is hosted on a free front-end cloud hosting service called Vercel.

Can I view Sonar's source code?

Yes. All code used to build our site is available to view on our Github repository.
